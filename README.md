##### Introduction
This application is developed with reactjs, material-ui, webpack and babel.

##### Installation
Requirements:
- git
- Node v7 or higher
- npm v4 or higher

To run the app locally:

```bash
git clone https://github.com/mikyaj/react-exercise.git
cd react-exercise
npm install
npm run start
open http://localhost:8080
```

To run the tests:

```bash
npm run test
```
